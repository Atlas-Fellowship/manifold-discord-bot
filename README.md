# Atlas Points

Atlas Fellowship Discord bot to track points.
<br/><br/><br/>

## Features
  * Leaderboard
  * Manage points by role (ex: `/add amount:100 role:@Group 1`)
  * Gift points to other users

## Requirements
  * Node.js
  * Yarn
